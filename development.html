GAME DEVELOPMENT

The first thing we did was establish the styles and mechanics of the game and leave them written down in a Google document. From there, we start a sprint and split all the functionalities into small tasks to make it easier to accomplish them.

After assigning each of us a task, we prepare our work environment and the programmers create a repository on GitHub and create a project in Unity while artists draw and model designs of ships and buildings. To link unity and github we have used a tool called Fork and that is how we have been controlling the versions of the game.

Hunt of the ore is a game where the map is generated procedurally as the ship moves and is destroyed when it leaves the ship's sight.
